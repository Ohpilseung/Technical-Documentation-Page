HTML syntax and structure

Html is a programming language that is used to develop webites.

All html codes begin with a !DOCTYPE html to indicate which version of html is been used.

After this, all other html code is nested within the html opening and closing tags.

Inside the html element, the code has a head and body elements.

The head element houses what we call the metadata such as the characters to use, the title to appear on the browser tab, the fonts to render the page and a link to a css file.

The body element contains the actual programing for rendering the website.

The html structure follows this format:

  1. < !DOCTYPE html >
  2. < html>
  3. < head>
  4. < head/>
  5. < body>
  6. < body/>
  7. < html/>
HTML elements

Html elements contain the starting tag, their content and a closing tag.

The tags are usually used to indicate what is to be rendered in the browser. It could be a text, button, input, image or a table.

The elements usually have an opening < tag > tag and a closing < tag /> tag. However, some elements are self-closing and would not have inidividual opening and closing tags < tag />.

Examples of html elements include:

  • h1
  • p
  • img
  • button
HTML ids and classes

Ids and classes are all used to mark or target elements for specific modification in css

Ids are more specific and can only be used to target only one specific element which does not reappear anywhere along the html.

The general way of targeting an element with an id is to include id="id-name" in the opening tag of the element.

Classes on the other hand are used to target several elements for a specific styling. They are less specific compared to ids.

They can target an element by including class="class-name" in the opening tag of that element.

CSS syntax and structure

CSS is also a programing language that is used to style html codes. It is basically used to style html elements.

CSS consists of selectors that are used to select specific elements, classes and ids for styling

CSS properties are also used to specify which portions of the html elements for styling while the values inidicate which style to apply

A typical CSS code consist of a selector with a curly brackets css-selector {}. Inside this curly bracket is the css-property : and the value;.

CSS types of selectors

There are several types of css selectors. These include:

  • element-selectors
  • #id-selectors
  • .class-selectors
  • and others.
